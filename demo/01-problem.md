# 1. Problem
About two years ago I started using AI to help me develop in Go but I held tight control of the code. But by Jan 2026 Claude had gotten so good I started just planning and letting it build. But that led to **_lots_** of waiting.

So I starting working on other projects during the wait; ones I've always wanted to work on but that were never a priority. I got to over 10 projects concurrently, but task switching was killing me and so were all the markdown files Claude generates.

A had an epiphany; build a tool to reign in all the chaos so I could stop loosing track of each project's status, and so I could stop constantly asking Claude to review and clean up all the markdown files used for planning.
