Resulting code shape (both functions):

```python
def project_config_path() -> Path | None:
    cwd = Path.cwd()
    for parent in [cwd] + list(cwd.parents):
        candidate = parent / ".endless" / "config.json"
        if candidate.exists():
            return candidate
    return None
```

Docstrings updated to reflect cwd-walk semantics and reference E-1140 (the decision reversing E-1112).
